# zedx

Boilerplate generator for Zed extensions.

## Installation

```bash
npm install -g zedx
```

### Usage

```bash
# Create a new extension
zedx

# Bump extension version
# Run inside extension dir
zedx bump patch   # 1.2.3 → 1.2.4
zedx bump minor   # 1.2.3 → 1.3.0
zedx bump major   # 1.2.3 → 2.0.0
```
